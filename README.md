# AI 時代語氣翻譯機

AI 時代語氣翻譯機 是一個結合生成式 AI 與人文社會觀點的互動式網站原型。使用者可以輸入一句日常語句，或從預設的快速體驗句子中選擇一句，系統會透過 AI 將句子轉換成不同語氣，並提供文化洞察，讓使用者看見語言背後的時代、媒介與社會文化意涵。

本專案不是單純的文字改寫工具，而是希望透過「語氣轉換」探索：同一句話在不同文化語境中，如何被重新理解與表達。

## Project Concept

本專案聚焦於「語氣」作為一種文化現象。

同一句話在不同語境中，可能呈現完全不同的情緒、身份與社會意義。例如：

我今天很累，不想上班。

這句話可以被轉換成古典文言文、校園匿名社群風、中英混用風、長輩風、台式口語風或賽博龐克風。每一種語氣不只是文字風格，而是對應到不同的文化場景、媒介形式與社會想像。

## Features
### 1. 快速體驗模式

使用者可以從預設句子中選擇一句，快速體驗 AI 語氣轉換。

目前預設句子包含：
 - 我今天很累，不想上班
 - 我只想躺平，但世界不放過我
 - 今天喝到好喝的飲料，覺得人生又可以了
 - 我以為我已經沒事了，結果聽到那首歌又破防了


### 2. 自由輸入模式

使用者可以自行輸入一句話，讓 AI 即時生成不同語氣版本。

### 3. 六種語氣轉換

目前支援六種語氣：
- 古典文言文
- 校園匿名社群風
- 中英混語風
- 長輩風
- 台式口語風
- 賽博龐克風

### 4. AI 文化洞察

系統會針對句子本身產生文化洞察，說明該句話反映的現代生活情緒、社會現象或文化意涵。

### 5. 語境插圖

快速體驗模式搭配語境插圖，讓使用者更直觀地理解每一句話背後的心境。

## Humanistic Perspective

本專案希望從人文社會角度思考生成式 AI 的應用，而不只是展示 AI 的文字生成能力。

本專案關注的問題包括：
- 語氣如何反映不同時代與媒介？
- 網路語言如何改變人們表達情緒的方式？
- 中英混語、家庭通訊、校園社群與台式口語如何反映不同文化場景？
- AI 在模擬語氣時，是否可能複製或強化刻板印象？
- 一句日常語言如何承載工作疲憊、躺平文化、小確幸與情緒破防等當代經驗？
